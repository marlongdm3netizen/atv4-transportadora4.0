# Plano de Execução (8h)

Desenvolvimento de Sistemas

SA04 – Organização do trabalho e definição simples de prazos para o protótipo

Equipe: Individual

Integrante:Marlon Mendes Mazarin

Projeto: Sistema de Compras e Transportadora


# 1. Objetivo

Organizar as tarefas restantes do protótipo utilizando um quadro Kanban, definindo prioridades, responsáveis, estimativas 
e prazos para execução durante duas aulas (8 horas totais).


# 2. Tarefas Must (Obrigatórias)

| ID | Tarefa                           | Prioridade | Estimativa | Prazo  |
|----|----------------------------------|------------|------------|--------|
| 01 | Criar página inicial de produtos | Must       | Média      | Aula 1 |
| 02 | Adicionar 9 produtos ao catálogo | Must       | Média      | Aula 1 |
| 03 | Desenvolver página de compra     | Must       | Média      | Aula 1 |
| 04 | validação do formulário          | Must       | Curta      | Aula 1 |
| 05 | Criar painel da transportadora   | Must       | Média      | Aula 2 |
| 06 | Implementar listagem de pedidos  | Must       | Média      | Aula 2 |
| 07 | Ajustar CSS e responsividade     | Should     | Curta      | Aula 2 |
| 08 | Elaborar README e documentação   | Must       | Curta      | Aula 2 |


# 3. Cronograma

Aula 1 (4 horas)

Planejado

- Criar página inicial
- Inserir produtos
- Desenvolver página de compra
- Implementar validação dos formulários

Resultado Esperado

Cliente consegue visualizar produtos e realizar pedidos.


Aula 2 (4 horas)

Planejado

- Criar painel da transportadora
- Exibir pedidos cadastrados
- Ajustar layout
- Criar documentação

Resultado Esperado

Sistema completo e documentado.


# 4. Estimativas

| Tarefa                   | Estimativa | Justificativa                       |
|--------------------------|------------|-------------------------------------|
| Página inicial           | Média      | Possui estrutura visual e navegação |
| Cadastro de produtos     | Média      | Inclusão de vários itens e imagens  |
| Página de compra         | Média      | Contém formulário e validações      |
| Validação JavaScript     | Curta      | Pequenas verificações               |
| Painel da transportadora | Média      | Interface administrativa            |
| Listagem de pedidos      | Média      | Exibição dos dados cadastrados      |
| Ajustes CSS              | Curta      | Melhorias visuais                   |
| README                   | Curta      | Documentação simples                |


# 5. Riscos e Ações Preventivas

| Risco/Dificuldade                | Ação Preventiva                                  |
|----------------------------------|--------------------------------------------------|
| Erro nos links entre páginas     | Testar navegação após cada alteração             |
| Problemas no JavaScript          | Verificar Console do navegador durante os testes |
| Imagens não carregarem           | Conferir caminhos das pastas antes da entrega    |
| Falta de tempo para documentação | Reservar parte da Aula 2 para o README           |
